template<int dim, int dimworld, typename ScalarType = double>
class Dune::IGA::NURBSGrid< dim, dimworld, ScalarType >

[ provides Dune::Grid ]

Provides a Grid with NURBS Geometry

YaspGrid stands for yet another structured parallel grid. It implements the dune grid interface for structured grids with arbitrary overlap (including zero),

Template Parameters
dimThe dimension of the grid
dimworldThe dimension of the grid's embedding space
ScalarTypethe field type (use float, double, complex, etc)
