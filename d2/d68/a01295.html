Detailed Description

template<int dim>
class Dune::Functions::NurbsLocalCoefficients< dim >

Attaches a shape function to an entity.

The attachment uses the same degree as for Qk elements. This does not provide sufficient information to compute global indices for the shape functions. However, it does allow to find all degrees of freedom that belong to the grid boundary, if the knot vector is open.

Note
Currently only implemented for 1d and 2d grids. For higher dimensions you can still use the BSplineBasis, but you won't be able to find the degrees of freedom on the grid boundary.
