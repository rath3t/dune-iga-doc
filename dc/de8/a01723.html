Detailed Description

template<int netdim, class ElementType, class Container = std::vector<ElementType>>
class Dune::IGANEW::MultiDimensionalNet< netdim, ElementType, Container >

This is a condensed version of the proposal https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2022/p1684r2.html It is a container with multidimensional accessors.

It has less features and only allows runtime size values but the rank is given at compile time. Furthermore, there are specific constructors to have a convenient construction for NURBS controlpoints It also has several overloaded operators to do linear algebra with it such as operator*

Template Parameters
ElementTypeThe type of the stored values
netdimThe rank of the array (How many dimensions it has)
ContainerThe type of storage
