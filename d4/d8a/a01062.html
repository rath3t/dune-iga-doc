Detailed Description

template<Concept::Vector VT>
struct Dune::IGANEW::ControlPoint< VT >

Represents a NURBS control point.

This struct stores the coordinates and weight of a NURBS control point. The coordinates are represented by the position vector p of type VectorType, and the weight is represented by the scalar w.

Template Parameters
VTThe type of coordinates of the control point position.
Note
The vector type must satisfy the Concept::Vector concept.
