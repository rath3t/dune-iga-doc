Detailed Description

template<int dim, int dimworld, typename ScalarType>
class Dune::IGANEW::DefaultTrim::Trimmer< dim, dimworld, ScalarType >

Trimmer struct representing a trimmer with identity trimming (no trimming at all).

DefaultTrimmer class representing a trimmer with reasonable defaults.

Template Parameters
dimDimension of the patch.
ScalarTypeScalar type for the coordinates.
dimDimension of the patch.
