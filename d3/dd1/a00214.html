Finds the closest point projection on a NURBS curve or surface by trust region method.

This function computes the closest point projection of a given point onto a NURBS curve or surface. The trust region method is used to iteratively refine the solution. The function returns a tuple containing the local coordinates of the closest point projection, the residual norm, the energy value, and the distance between the projected point and the original point.

Template Parameters
GeoThe NURBS geometry type
Parameters
geoThe NURBS geometry object.
pointThe point to be projected onto the NURBS geometry.
startOptional starting point for the iteration. If not provided, the midpoint of the domain is used.
Returns
A tuple containing the local coordinates, residual norm, energy value, and distance from the point.
Exceptions
Dune::MathErrorif the trust region method fails to find an energy-decreasing direction.

Finds the intersection point of two 3D lines.

This function calculates the intersection point of two 3D lines defined by their basepoints and direction vectors. The lines are represented by the equations:

Line 1: \(P_1(t) = \text{{basepoint1}} + t \cdot \text{{direction1}}\)

Line 2: \(P_2(s) = \text{{basepoint2}} + s \cdot \text{{direction2}}\)

where \(t\) and \(s\) are parameters. The function returns the point where the two lines intersect.

Template Parameters
VectorTypeA vector type satisfying the Concept::Vector concept.
Parameters
basepoint1The basepoint of the first line.
direction1The direction vector of the first line.
basepoint2The basepoint of the second line.
direction2The direction vector of the second line.
Returns
The intersection point of the two lines.
Exceptions
Dune::MathErrorif the lines are almost parallel (within tolerance).

Create a surface of revolution from a generating curve.

This function generates a NURBS surface of revolution based on a given generatrix curve, origin point, revolution axis, and angle. The generatrix is a 1D NURBS curve that is revolved around the specified axis.

Template Parameters
ScalarTypeThe field type for coordinates (e.g., float, double, complex).
Parameters
generatrix1D NURBS curve representing the generatrix to be revolved.
pointThe origin point of revolution.
revolutionAxisThe axis of revolution.
revolutionAngleThe angle by which the generatrix should be revolved in degrees (default is 360 degrees for a full revolution).
Returns
NURBSPatchData representing the generated surface of revolution.
Note
The angles are specified in degrees.

Example (Create a torus):

using ScalarType = double; // or any other numeric type
auto generatrix = makeCircularArc<ScalarType>(1.0, 0.0, 180.0); // Full circle with radius one
auto torus = makeSurfaceOfRevolution<ScalarType>(generatrix, {2, 0, 0}, {1, 0, 0});
