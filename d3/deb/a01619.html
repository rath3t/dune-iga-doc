Detailed Description

template<int mydim, int coorddim, class GridImp>
class Dune::IGANEW::PatchGridLocalGeometry< mydim, coorddim, GridImp >

This class describes entities in the 0..1 space if untrimmed this is in the cube, for the untrimmed case this is simply a wrapper around the Yasp grid local geometry.
